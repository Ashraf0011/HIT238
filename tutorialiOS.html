How I installed Cordova for iOS

.....

Step 1: Install Node.js

First of all we need to install Node.js on the system. It can be downloaded from Node.js Official Website It will install npm on the system. Once done with that, we can access npm commands on terminal.

Step 2: Installing Cordova with npm

There are many tutorials online whi are not really very help full. In my case, i had trouble with following tutorials at had some issues with read and write permission on my mac. But i figured out that using sudo command solved the issues for me. 

            
              
                $ sudo npm install -g cordova
It will require account password for mac.

Step 3: installing ios simulator

Once Cordova is installed on system we hav to install ios-simulator. If it not installed a bad fortune will be coming very soon ü§£ü§£ü§£ü§£!!! Anyway, to install ios simulator we just run following commands on terminal. 

            
              
                $ sudo npm install -g ios-sim
Once again it will ask for password.
Please make sure to use "-g" with commands it will install features globally, meaning, we will be able to run commands from any directory.
So far that's all we need. we can now create cordova project.

Step 4: Create Cordova Project.
